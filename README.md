# Modern Product Handbook

A practical, organisation-agnostic guide to building and improving products in an AI-enabled landscape. Use it to onboard teammates, calibrate product craft, and align cross-functional teams around outcomes.

## Read the handbook

- [Product principles & philosophy](./principles/README.md)
- [Product lifecycle & practices](./lifecycle/README.md)
- [Product competency model](./competencies/README.md)
- [Tools & templates](./templates/README.md)

## How to use this repo

- Start with the principles to align on mental models.
- Use the lifecycle as an operating playbook for day-to-day work.
- Use competencies for self-assessment, coaching, and growth plans.
- Use templates to apply the handbook consistently.
